## Project Overview
**Paradise Nursery** is a dynamic web application built for an online plant shop. The application allows users to browse a curated selection of house plants, view specific details, and manage a functional shopping cart.

The project emphasizes interactive UI/UX, state management for the shopping cart, and real-time cost calculations.

## ğŸš€ Features

### Product Browsing
* **Plant Landing Page:** A visually appealing gallery displaying available plants.
* **Product Details:** View high-quality images, scientific/common names, descriptions, and unit prices for each plant.

### Shopping Cart Functionality
* **Add to Cart:** Users can add plants to their cart directly from the browsing page.
* **Dynamic Cart Management:**
    * **Increment/Decrement:** Adjust the quantity of specific items within the cart.
    * **Remove Items:** Delete items entirely from the cart.
* **Real-Time Calculations:** The total cost updates instantly as items are added, removed, or quantities are adjusted.

### User Interface
* **Navigation Bar:** A persistent, responsive navigation bar that includes a cart icon with a dynamic badge showing the total number of items selected.
* **Responsive Design:** Optimized for desktop and mobile viewing.
